# AWS S3 Backup Automation

This project automatically backs up files to an AWS S3 bucket using Python and GitHub Actions CI/CD pipeline.

## Features
- Upload files to AWS S3
- Automated backups using cron
- CI/CD pipeline with GitHub Actions

## Technologies Used
- Python
- AWS S3
- GitHub Actions
- Linux

User → GitHub → GitHub Actions → AWS S3

## Future Improvements
- Add email notifications
- Add multiple file support
